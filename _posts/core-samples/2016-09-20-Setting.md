&#160; &#160; &#160; &#160; 基本上讲完了，当然几本上什么也没有讲，因为原作者就是这么配置的，我们只需要跟随巨人的脚步即可。
但是还有一些与Github相关的配置，因为博客毕竟是要放在Github上的。  

### 你可以在Github上修改的配置  

```
 github: [metadata]
 kramdown:
   input: GFM
   hard_wrap: false
 gems:
   - jekyll-coffeescript
   - jekyll-paginate
```  

### 你不可以在Github上修改的配置  

```
 lsi: false
 safe: true
 source: [your repo's top level directory]
 incremental: false
 highlighter: rouge
 gist:
   noscript: false
 kramdown:
   math_engine: mathjax
```
